Готово! Как мне переконвертировать данные обратно в строку?

Для этого потребуется другой объект модуля datetime, а именно - метод isoformat().

result = latest.isoformat()
Отлично, как должна выглядеть моя программа?

Вы должны поместить свой код в заданную функцию и, связав все предыдущие подсказки, получить итоговый результат. 

from datetime import datetime


def get_latest(dt1str: str, dt2str: str) -> str:

    dt1obj = datetime.fromisoformat(dt1str)
    dt2obj = datetime.fromisoformat(dt2str)

    latest = dt2obj if dt2obj > dt1obj else dt1obj
    result = latest.isoformat()

    return result

Так, теперь давайте попробуем оптимизировать это решение. Во-первых, вы можете использовать кортежи и встроенную функцию map() что бы получить объекты datetime. 

dt1obj, dt2obj = map(datetime.fromisoformat, (dt1str, dt2str))

Во-вторых, сделать сравнение, используя встроенную функцию max() и сразу перевести из даты в строку. Давайте посмотрим на код еще раз: 

from datetime import datetime


def get_latest(dt1str: str, dt2str: str) -> str:

    dt1obj, dt2obj = map(datetime.fromisoformat, (dt1str, dt2str))
    result = max(dt2obj, dt1obj).isoformat()

    return result

Более того, если вы абсолютно уверены, что все строки имеют одинаковый формат, вы можете просто их сравнить "как есть" не преобразовывая в формат даты, поскольку Python, сопоставляя каждый символ попарно друг другу, сразу найдет более позднюю дату. 

def get_latest(dt1str: str, dt2str: str) -> str:

    return max(dt1str, dt2str)
